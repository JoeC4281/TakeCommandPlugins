# PERMUTE

PERMUTE n \[/N\|/S\] - generate a random permutation of 0 ... N-1

&nbsp;

/N - newline-separated; /S - space-sepatered;

&nbsp;

defaults: single string (n \<= 10), space-separated (n \> 10)

&nbsp;

PERMUTE's output is not terminated by a space or a newline
